# JoomlaTranslate

Official translation repository for JoomlaTranslate.

This repository contains Joomla extension language files that are translated and maintained through the JoomlaTranslate platform.

## Repository Structure

```
administrator/
└── language/
    ├── en-GB/
    └── fa-IR/

language/
├── en-GB/
└── fa-IR/
```

## Features

- Official Joomla extension language files
- AI-assisted translations
- Multi-language support
- GitHub-based version control
- Translation progress tracking

## Supported Languages

Translations are available for multiple Joomla languages. New languages are added continuously.
